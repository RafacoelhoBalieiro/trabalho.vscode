Cryptojacking

-O Cryptojacking é quando cibercriminosos usam sem autorização um dispositivo de outra pessoa para mineirar criptomoedas

-Ele acontece quando o invasor instala no dispositivo um malware ou executa scripts escondidos, que normalmente são via sites, que usam o poder de processamento da vítima.

-O objetivo dele é gerar lucro para o criminoso minerando criptomoedas como Monero, sem custos próprios de energia ou hardware.
